# Clover

> Make lover closer.

基于 [Kaimou985/lovelist](https://github.com/Kaimou985/lovelist) 产品思路重新设计的恋爱清单 PWA。

## 品牌

`Clover` 取自 “make lover closer”。Logo 由四片相互靠近的心形叶片组成，珊瑚色代表心动，鼠尾草绿代表陪伴与成长。

## 功能

- 分类整理的 100 件恋爱小事、搜索、筛选和收藏
- 完成日期、地点、心情、文字与照片记录
- 回忆时间线
- 月度约会账本、分类统计、预算进度
- 情侣信息、纪念日、三套主题
- JSON 完整备份与严格恢复校验
- localStorage 本地持久化、Service Worker 离线缓存
- 响应式桌面工作台与移动端 App 导航

## 本地运行

```bash
pnpm install
pnpm dev
```

打开 `http://localhost:3000`。
